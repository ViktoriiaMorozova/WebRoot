Rent a House: BNB-B project

A bed and breakfast alternative, the BNB:B app allows customers to search for available properties worldwide. Renting homes has never been easier and more fun with our innovative app.

Our app features a home view with a search function to let customers find rentals using building data search terms. Once the customer enters all necessary information, the system uses API data to display results that match their search criteria. A customer can then choose a property, click on a card, and receive all the information they need. At that point, the customer can hit the apply button. After entering the application view, the user will fill out the form and submit it. Our app is designed for desktops, tablets, and mobile phones and runs in a web browser. Our data are stored locally on the browser and pulled from an online property database using an API. Information such as prices, numbers of rooms, etc., will be returned by our JSON objects.
